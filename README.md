Based off of [Docker Selenium](https://github.com/SeleniumHQ/docker-selenium) chrome base image.

Following softwares are ready to use in this image:

- Python3
- Pip3
- Selenium for Python
- Chrome (Headless)
- XvFB (Non-Headless chrome)
